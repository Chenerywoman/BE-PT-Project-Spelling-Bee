Spelling Bee: API routes

api/years

  • GET /api/years
  • Returns an object with an array of available years.
  • GET /api/years/:year
  • Returns an object containing the requested year.

api/categories

  • GET /api/categories
  • Returns an object with an array of categories with the name, description & years the categories belong to.
  • GET /api/categories?year={year}
  • Returns an object with an array of categories which belong to a particular year.
  • GET /api/categories/:category
  • Returns an object containing the requested category.

api/words

  • GET /api/words
  • Returns an object containing an array of word objects which each contain the word and the year(s) it belongs to. Each word object also contains a 'partials' array. (A partial is part of a word - suffix, prefix or medial - see below). Some words have no partials, so the partials array is empty. If a word contains a partial, e.g. 'superman' has the prefix 'super', the partials array contains the partial ('super') in the key 'letters' and the category it belongs to, e.g. prefixes.
  • POST /api/words
  • Posts a word object to the database. This requires a JSON body with the following structure: {"word": "pineapple", "partial": "pine", "year": 3}
  • DELETE /api/words
  • Deletes a word object from the database. This requires a JSON body with the following structure: {"word": "pineapple"}

api/prefixes

  • GET /api/prefixes
  • Returns an object with an array of prefix objects, containing the letters of the prefix, the years and category it belongs to.
  • GET /api/prefixes?prefix={prefix}
  • With a prefix query e.g /api/prefixes?prefix=anti Returns an object with an array of word objects where the prefix in the query string is in the word's partials array.

api/suffixes

  • GET /api/suffixes
  • Returns an object with an array of suffix objects, containing the letters of the suffix, the years and category it belongs to.
  • GET /api/suffixes?suffix={suffix}
  • With a suffix query e.g /api/suffixes?suffix=ly Returns an object with an array of word objects where the suffix in the query string is in the word's partials array.

api/medials

  • GET /api/medials
  • Returns an object with an array of medials objects, containing the letters of the medial, the years and category it belongs to.
  • GET /api/medials?medial={medial}
  • With a medial query e.g /api/medials?medial=sc Returns an object with an array of word objects where the medial in the query string is in the word's partials array.

api/homophones

  • GET /api/homophones
  • Currently under development - currently returns a description of homophones.
  • GET /api/homophones?homophone={homophone}
  • Currently under development.

api/freestyle

  • GET /api/freestyle
  • Returns an object with an array of words where the partials array is empty, i.e. it does not contain any listed prefixes, suffixes or medials.
